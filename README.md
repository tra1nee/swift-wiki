# 🦊 Swift Вики

Личный справочник по **Swift / iOS** с мгновенным поиском, прогресс-дашбордом и графиком активности (как на GitHub).

🔗 **Открыть:** https://tra1nee.github.io/swift-wiki/

- `index.html` — приложение (поиск, блоки по фазам, видео-ссылки, дашборд: журнал занятий, стрик, прогресс-бары).
- `data.js` — контент: 75 карточек по фазам (Swift → SwiftUI → сеть/данные → архитектура → UIKit → инструменты → найм + база разработчика).

Прогресс и журнал по умолчанию хранятся локально в браузере. Можно включить **синхронизацию между устройствами** через вход Google (Firebase) — данные приватны для каждого аккаунта. Инструкция: [SETUP.md](SETUP.md).

Часть личной учебной платформы для перехода в iOS-разработку.
